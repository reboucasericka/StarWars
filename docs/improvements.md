# Lista de Melhorias Aplicadas

1. **Barra de Pesquisa**:
   - Adicionada para permitir buscas dinâmicas de filmes e personagens.
   - Implementada no front-end utilizando JavaScript.

2. **Logs no Banco de Dados**:
   - Todas as requisições à API são registradas no banco de dados para análise futura.
   - Estrutura do banco inclui campos como ID, endpoint acessado, método HTTP e timestamp.

   - O projeto foi organizado utilizando o padrão Model-View-Controller, tornando-o mais modular e escalável.

5. **Configuração de Rotas Amigáveis**:
   - Utilização do arquivo `.htaccess` para limpar e padronizar URLs.

6. **Responsividade e Design**:
   - Melhorias no layout para tornar o site acessível em dispositivos móveis e desktops.
   - Design mais intuitivo, com elementos estilizados em CSS.
